# Development roadmap

## Phase 1 — Project setup
- Scaffold React/Vite frontend and Express/TypeScript backend
- Add shared tooling and environment configuration

## Phase 2 — Database schema
- Define Prisma models for users, organizations, subscriptions, plans, invoices, notifications, files, and logs
- Create migration files and seed data

## Phase 3 — Authentication and authorization
- Register, login, refresh flow, password reset, email verification
- Implement RBAC and protected routes

## Phase 4 — User management and admin panel
- List, filter, search, and sort users
- Admin-only controls for role and status changes

## Phase 5 — Core product modules
- Settings, notifications, activity logs, file uploads, reporting
- Responsive dashboard experience

## Phase 6 — Subscription system
- Pricing plans, trial mode, monthly/yearly selection, upgrade/downgrade, invoices
- Billing history and export-ready reporting

## Phase 7 — Hardening and delivery
- Testing, logging, security review, documentation, deployment configuration

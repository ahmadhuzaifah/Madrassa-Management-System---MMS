# User guide

## Getting started

1. Register a new account.
2. Verify your email when prompted.
3. Select a plan from the billing screen.
4. Explore dashboard, settings, reports, uploads, notifications, and activity history.

## Admin features

- View and manage users from the Users screen
- Review audit logs and reports
- Manage plans and subscriptions
- Manage workspace members and settings

## Common workflows

- Update profile settings from Settings
- Upload files from the Files section
- Review alerts from Notifications
- Export invoices from Reports
- Change password from Profile
- Review recent activity from Activity
